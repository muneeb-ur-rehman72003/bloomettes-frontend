# Creative Portfolio & Shop Frontend Website

A responsive creative portfolio and shop frontend website built using HTML5, CSS3, and JavaScript.
The website includes artwork galleries, poetry/book sections, handmade product cards, blog layouts, testimonials, and a clean visual design focused on creativity and user experience.
